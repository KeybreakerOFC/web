## 🎯 Sobre este repositorio

Este repositorio está **completamente libre** para que cualquiera lo use como **base, guía o template** para sus propias webs.

Puedes:
- Clonarlo
- Modificarlo
- Usarlo en proyectos personales o comerciales
- Subirlo a Vercel, Netlify, Hostinger, etc.

**No es necesario dar crédito**, aunque siempre es bienvenido 🔥

## ✨ Características

- Diseño **cyberpunk / hacker / red team**
- Cursor personalizado con efecto de seguimiento
- Efectos glitch y scanlines
- Totalmente responsive (PC + Móvil)
- Colores: Negro, Rojo intenso y Blanco
- Optimizado para velocidad
- Fácil de personalizar

- HTML5
